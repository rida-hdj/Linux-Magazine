كيف تقسم بارتشنز

كيفية تقسيم البارتشنز . خطوة مهمة جدا إذا كنت بتثبت توزيعة مثل ارش

أوامر

Nothing: الناشر

السلام عليكم ورحمة الله وبركاته, طبعا لو كنت ثبتت آرش او جنتو او اي توزيعة تعتمد على المستخدم بالتثبيت اليدوي اكيد تعرف كيف تقسم بارتشنز ووش الادوات اللي بتستخدمها, من الأدوات السهلة لتقسيم البارتشنز وبواجهة TUI يعني تقدر تستخدم الكيبورد في استخدامها هي cfdisk ومتوفرة تقريبا بكل البيئات الحية لتوزيعات لينكس وموجودة بشكل اساسي في كل التوزيعات المهم نبدأ بكيف نستخدمها وطريقة تقسيم البارتشنز

طريقة استخدام CFDISK وتقسيم القرص:

اول شيء تحدد القرص اللي عندك واللي تبغى تقسمه

$ lsblk
NAME MAJ:MIN RM SIZE RO TYPE MOUNTPOINTS
sda 8:0 0 298.1G 0 disk

التقسيمة اللي بنسويها بتكون التقسيمة البدائية الطبيعية

UEFI:

  • Swap: بنسويها 4GB في هذا الشرح
  • EFI: 512MB
  • Root: باقي المساحة المتبقية

BIOS:

  • Swap: 4GB
  • EFI: 512MB
  • Root: باقي المساحة المتبقية

يلا نبدأ , الحين بنستخدم CFDISK

$ sudo cfdisk /dev/sda

بتطلع لك واجهة تختار فيها GPT لو جهازك UEFI وبتختار DOS لو جهازك BIOS المهم بعدها تختار القرص تضغط عليه وتكتب المساحة اللي بتكون لل EFI اللي بتكون 512MB (لاجهزة UEFI فقط) وبعدها تروح (بالأسهم) لكلمة TYPE وتختارها وتختار نوع EFI System , ذا بيضمن ان Bootloader يستخدم بارتشن EFI عشان يقلع منه , بتكتب 512M بخانة السؤال عن الحجم ذا بيكون /dev/sda1 بعدها نختار القرص الاساسي مرة ثانية بالضغط عليه بزر ENTER وبعدها تكتب مساحة الSwap اللي بتكون 4GB ورح تكتب 4G وتختار كلمة TYPE وتحدد Linux Swap / Solaris ذا بيكون /dev/sda3 عشان مساحة الروت بنسوي نفس الشيء لكن ما بنحدد حجم فقط بنضغط ENTER, وذا بيكون /dev/sda2 , طبعا اجهزة BIOS رح تخلي الروت BOOTABLE فقط رح تختار بارتشن الروت ورح تختار BOOTABLE وكذا تكون خلصت التقسيمة بتختار WRITE وبتكتب yes وكذا التقسيمة تمت (استخدم lsblk عشان تتأكد من التقسيمة)

تحديد نوع البارتشنز:

قسمت البارتشنز؟ يلا نحدد نوعهم (او نفرمتهم Formatting)

UEFI:

$ sudo mkfs.fat -F 32 /dev/sda1
sudo mkfs.ext4 /dev/sda2
sudo mkswap /dev/sda3

# Enable Swap
sudo swapon /dev/sda3

BIOS:

$ sudo mkfs.ext4 /dev/sda1
sudo mkswap /dev/sda2

# Enable Swap
sudo swapon /dev/sda2
